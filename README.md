[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Amankushwaha2005/Ankit-portfolio)

## Local Run

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Open: http://localhost:5000

## Stack

- HTML / CSS / JavaScript
- Python Flask backend
- Contact form + Careers resume upload
